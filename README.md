# rust-molar-mass

A very simple tool to calculate the molar mass of different molecules.

### Usage 

* compile and run main.rs
* input molecules

### Example

>> C6H12O6 CO2 CH4
C6H12O6    :   180.156 g/mol
C1O2       :   44.009 g/mol
C1H4       :   16.043 g/mol
>> Zn2
Zn2        :   130.76 g/mol
